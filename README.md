# 흥할지도 (HeungMap)

> **정확한 수요 예측으로 기획자와 방문객 모두를 만족시키는 데이터 기반 축제 플랫폼**
> 한국관광공사 TourAPI(축제·관광 정보)와 지역별 방문자 빅데이터를 학습한 예측 모델로,
> **기획자**에게는 예상 방문객 수 예측 → 개최지·주차·숙박 등 편의시설 최적 구성과 기획안·보완 피드백을,
> **방문객**에게는 예상 혼잡도·티켓팅 경쟁도와 주변 주차·숙소 정보 등 축제를 더 즐길 수 있는 방법을 제안합니다.

- **팀명**: 경사하강단
- **서비스명**: 흥할지도 (영문 `HeungMap`) — "흥할지(도) 몰라" + "지도(map)"
- **공모전**: 2026 관광데이터 활용 공모전 ②-2 웹·앱 구현 부문 (한국관광공사) / 지정과제 **9번**
- **팀 구성**: 2인 (개발 초심자 — ML/데이터파이프라인/풀스택 첫 도전)

## 이게 뭔가요 (30초 요약)
축제 흥행을 **주관적 경험이 아니라 데이터로 예측**하는 게 핵심. 하나의 예측 엔진이 양쪽을 먹입니다.
- **기획자 모드**: 축제 조건(지역·시기·규모·테마 등) 입력 → 예상 방문자 수/흥행 점수 + 근거 + 개최지·편의시설 추천 + 보완 리포트
- **방문객 모드**: 등록된 축제 탐색 → 예상 혼잡도·티켓팅 경쟁도 + 주변 주차/숙소/연계 관광지 + 방문 코스 추천
